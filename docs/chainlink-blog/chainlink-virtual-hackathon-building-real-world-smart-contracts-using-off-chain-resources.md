为了帮助您最大限度地提高项目的成功，我们将为参与者提供直接接触团队成员和经验丰富的导师，大量的教育材料，学习外链连接，现场研讨会展示如何使用 Chainlink，等等。加入我们的 [Discord](https://discord.gg/aSK4zew) 来认识其他黑客，今天就开始计划吧！

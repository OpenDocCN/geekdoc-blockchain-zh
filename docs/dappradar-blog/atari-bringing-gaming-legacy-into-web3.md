# 雅达利将游戏遗产带入 Web3

> 原文：<https://web.archive.org/web/https://dappradar.com/blog/atari-bringing-gaming-legacy-into-web3>

## 雅达利正通过雅达利将其游戏遗产带入 Web3。X

**雅达利。x 是这家传奇游戏公司的新 Web3 分支，导演泰勒·德雷维茨希望带回雅达利的合作遗产。由于其广泛的 NFT 品牌网络，他们在雅达利 NFT 贸易公司的销售额总计超过 100 万美元。在与 DappRadar 的 AMA 中，他们解释了他们对 Web3、元宇宙和区块链游戏未来的所有愿景**

雅达利在 80 年代初达到巅峰，但成为 80 年代初视频游戏崩溃的象征。在任天堂进入市场之前，没人指望家用游戏机会再次成功。一晃 40 年过去了，雅达利成立了一家名为 Atari.X 的 Web3 公司。

该团队相信，区块链将继续成长为他们业务的重要组成部分，并为他们创造新的场所，与他们的社区进行互动和协作。

这家标志性的游戏公司选择了沙盒来构建它的体验。拥抱用户生成的内容和元宇宙的沉浸式可能性。
